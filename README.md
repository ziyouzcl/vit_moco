# vit_moco

基于 3D CT 的 Vision Transformer 自监督表征学习框架。

本项目主要面向 3D CT 体数据，构建了一套从医学影像预处理、自监督预训练到下游分类微调的完整深度学习流程。项目重点关注 3D 医学影像在数据读取、ROI 定位、标准化输入、视觉表征学习和预训练权重迁移中的关键问题。

---

## 项目简介

`vit_moco` 是一个用于 3D CT 影像表征学习的自监督预训练框架。项目将 Vision Transformer 扩展到 3D 医学影像场景，并结合 MoCo v3 对比学习思想，使模型能够在无标注或少标注 3D CT 数据中学习有效的影像表征。

该框架主要包括：

- 3D CT 体数据读取与预处理
- 基于 ROI 的病灶区域定位与 3D patch 裁剪
- 适用于 3D 医学影像的 Vision Transformer 骨干网络
- MoCo v3 风格的自监督对比学习预训练
- 预训练权重迁移与下游分类微调
- YAML 配置化实验管理与统一训练入口

---

## 主要功能

### 1. 3D CT 数据预处理

项目支持对 3D CT 体数据进行标准化处理，包括：

- CT 图像读取
- 强度归一化
- ROI mask 定位
- 病灶质心与包围盒计算
- 固定尺寸 3D patch 裁剪
- 病灶区域标准化输入构建

通过上述流程，可以将不同尺寸、不同空间范围的 3D CT 数据转换为适合深度学习模型输入的统一格式。

---

### 2. 3D Vision Transformer

项目将传统 2D Vision Transformer 扩展至 3D 医学影像任务，构建适用于 CT 体数据的图像表征学习骨干网络。

主要特点包括：

- 3D patch embedding
- 支持不同 patch 尺度
- 支持 3D 卷积特征提取模块
- 适配 3D CT 体数据的空间结构建模

该部分使 ViT 能够处理 3D 医学影像中的体素空间信息，而不仅局限于二维图像特征。

---

### 3. MoCo v3 自监督预训练

项目基于 MoCo v3 思想搭建 3D 医学影像自监督预训练框架。

核心流程包括：

- 对同一 3D patch 生成双增强视图
- 使用在线编码器与动量编码器进行对比学习
- 通过对比损失学习稳定的 3D 影像表征
- 支持无标注 3D CT 数据上的预训练

该模块可用于在标注样本有限的医学影像场景中提升模型的特征提取能力。

---

### 4. 预训练权重迁移与下游微调

项目支持将自监督预训练得到的权重迁移至下游分类任务，包括：

- 加载 MoCo 预训练权重
- 提取并加载 ViT backbone 参数
- 替换下游任务分类头
- 支持冻结 backbone 的线性探测
- 支持端到端微调

该流程实现了从自监督表征学习到具体医学影像分类任务的完整闭环。

---

### 5. 配置化实验管理

项目使用 YAML 文件进行实验配置管理，统一组织：

- 数据集路径
- 模型参数
- 训练参数
- 优化器设置
- 学习率调度
- checkpoint 保存
- 日志记录

通过配置化管理，可以提高实验复现性和工程可维护性。
